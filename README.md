# Year 4 COM661 Coursework

## Description

Code used for the Full-stack Strategies and Development module's CW1 coursework submission.

## Activate venv: 
`venv\Scripts\activate`

## Updating requirements.txt: 
`pip freeze > requirements.txt`

## Run server: 
`python app.py`

## Deactive venv: 
`deactivate`
